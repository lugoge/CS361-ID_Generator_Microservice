# CS361-ID_Generator_Microservice
main collaborator: Gerardo Lugo

## Microservice Information
This microservice allows developers to generate a random id to utilize for their application.

## How to Use

### Requirements
Users must first run the requirements.txt file to download FLASK.
```
pip install -r requirements.txt
```
or 
```
pip3 install -r requirements.txt
```

### Run service
Run the microservice using the following commande:
```
python app/id_service.py
```
or
```
python3 app/id_service.py
```

### Success when the following message appears
Running on http://127.0.0.1:5000

## Users can test by using cURL in a new terminal
After running the application, in a new terminal run the following test file.
```
python test.py
```
or
```
python3 test.py
```
